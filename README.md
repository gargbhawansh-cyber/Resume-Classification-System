# Resume Classification System

This project classifies resumes into job categories using Machine Learning and NLP techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Naive Bayes

## Features
- Resume text preprocessing
- Resume classification
- TF-IDF vectorization
- Accuracy evaluation

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

## Output

- Predicts resume category
- Displays model accuracy
